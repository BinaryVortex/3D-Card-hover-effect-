# 🎴 3D Card Hover Effect

A modern **3D Card Hover Effect** built with **HTML, CSS, and
JavaScript** that creates an immersive mouse-tracking interaction. As
users move their cursor across the card, it tilts naturally to create
depth while delivering a smooth and responsive user experience.

## ✨ Features

-   🖱️ Interactive mouse-tracking 3D tilt
-   🎨 Modern, clean UI
-   ⚡ Smooth CSS transitions
-   📱 Responsive design
-   🪶 Lightweight with no external frameworks
-   🔧 Easy to customize

## 🛠️ Technologies

-   HTML5
-   CSS3
-   JavaScript (ES6)

## 🚀 Getting Started

1.  Clone the repository:

``` bash
git clone https://github.com/BinaryVortex/3D-Card-Hover-Effect.git
```

2.  Open the project folder.

3.  Launch `index.html` in your browser.
